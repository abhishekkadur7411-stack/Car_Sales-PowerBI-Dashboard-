# Car_Sales-PowerBI-Dashboard-
Power BI dashboard analyzing car sales data
Project Overview:
This project focuses on analyzing car sales data using Power BI to uncover trends, performance patterns, and business insights. The dashboard provides an interactive view of sales across different regions, brands, and time periods.

Dataset:-
  Source: Provided dataset
  Contains:Car brands
  Sales revenue
  Units sold
  Regions
  Time (monthly/yearly data)

Key Insights:
The top-performing car brand is [ACURA,AUDI]
The region generating the highest revenue is [ACURA]
Sales show a noticeable trend of [BMW]
Certain brands perform better in specific regions, indicating market preference differences

Dashboard Features:
Sales trend over time
Revenue comparison by brand
Region-wise performance
Interactive filters for detailed analysis

🛠 Tools Used
Microsoft Power BI
